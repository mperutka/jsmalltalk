JSmalltalk je impementace jazyka Smalltalk-80 v Javascriptu. Funkčně se snaží být JSmalltalk co nejvíce podobný prostředí Squeak. JSmalltalk je publikován pod licencí GNU GPL.

Integrované vývojové prostředí (IDE) JSmalltalku běží v prostředí prohlížeče a je postaveno na frameworku Ext JS (verze 3). Dále využívá skvělý editor CodeMirror. IDE zahrnuje class browser, workspace, message names, changes viewer, object inspector a debugger, workspace zatím chybí (lze ale využít javascriptovou konzoli pomocí příkazu Console>>log:).

První verze vznikla v roce 2010, v roce 2011 došlo k zásadnímu přepracování systému tříd a metatříd (viz jst-core.js). Velkou inspirací pro JSmalltalk byl obdobný projekt Amber Smalltalk, zejména řešení posílání zpráv, implementace thisContext a "objev" editoru CodeMirror.

JSmalltalk je vyvíjen v Google Chrome, kde funguje nejlépe (a nejrychleji) včetně ukládání provedených změn pomocí lokálního File System API. Ještě je testován ve Firefoxu, kde je v podstatě funkční (ovšem pomalejší než v Chrome a zatím stále bez File API). Na rychlost má značný vliv zapnutí/vypnutí ladícího módu (přítomnost/nepřítomnost souborů jst-debug.js a jst-tools-debugger.js). V ladícím módu se vytvářejí kontexty metod a bloků, což přináší zpomalení a vyšší nároky na paměť.
